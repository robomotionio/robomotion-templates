# Future Tools Keyword Search

Future Tools' keyword search is the primary way users discover AI tools on the platform, returning relevant results from its catalog of thousands of products. When you search Future Tools for a keyword like "writing assistant," "image generator," or "code review," the results page displays a grid of matching tools, each showing the tool's position in results, name, a brief description, primary category, community upvotes, and a thumbnail image. This search-first approach lets you see the competitive density for any AI use case at a glance.

## What it extracts

- **Position** — Ranking position in the search results.
- **Tool Name** — AI product name.
- **Tool Link** — Direct link to the tool's Future Tools listing page.
- **Description** — Brief product overview.
- **Image** — Tool logo or thumbnail image.
- **Category** — Primary AI category classification.
- **Upvotes** — Community upvote count or rating indicator.
