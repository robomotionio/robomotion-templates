# YouTube Channel Info

YouTube hosts over 100 million active channels, and channel-level data is essential for influencer marketing, competitive analysis, and creator research. But YouTube's public pages only show a subset of data, the official API has strict quotas, and copying information from channel pages manually is impractical when you are evaluating dozens or hundreds of creators.

## What it extracts

- **Channel Name** — The display name of the YouTube channel.
- **Channel ID** — The unique identifier assigned to the YouTube channel.
- **Subscribers** — The number of subscribers shown on the channel page.
- **Videos** — The number of uploaded videos on the channel.
- **Logo** — The channel's profile picture or logo image.
- **Description** — The channel's About section text.
- **Joined date** — When the channel was created or joined YouTube.
- **Total view** — The cumulative view count across all channel videos.
- **Position** — The ranking or position of the channel in results.
- **Link Title** — The title text of linked social media profiles.
- **Link URL** — The URL of connected social media accounts.
