# Common Packages

| Package | Use Case | Key Nodes |
|---------|----------|-----------|
| `Core.Programming` | Data transform, loops | `ForEach`, `Function`, `Debug`, `Sleep` |
| `Core.Browser` | Web automation | `Open`, `OpenLink`, `ClickElement`, `GetValue`, `RunScript`, `Close` |
| `Core.FileSystem` | File operations | `Create`, `PathExists`, `WriteFile`, `Delete` |
| `Core.Excel` | Spreadsheet | `Open`, `CreateSheet`, `SetRange`, `Save`, `Close` |
| `Core.Net` | HTTP requests | `HttpRequest` |
| `Core.Flow` | Flow control | `Label`, `GoTo`, `Stop`, `Log` |
| `Core.Trigger` | Start triggers | `Inject` |
| `Core.Vault` | Credentials | `GetItem`, `ListItems` |

For full package documentation, use `get_llms_txt(namespace)`.
For node schemas, use `get_node_cards(nodeTypes[])`.
