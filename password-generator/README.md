# Password Generator

Generates a random alphanumeric password and copies it straight to your clipboard.

## How It Works

The flow creates a random password using uppercase letters, lowercase letters, and digits at the length you specify. The result is automatically copied to the system clipboard and also displayed in a dialog box so you can see it.

## Configuration

Open the **Config** node and set:

| Variable | Description | Example |
|---|---|---|
| `msg.passwordLength` | Number of characters | `16` |
