# Freelancer.com Project Details

Freelancer.com is one of the world's largest freelance marketplaces, hosting millions of projects across software development, design, writing, marketing, data entry, and dozens of other categories. Each project page contains rich data: the project description outlining requirements, budget range or fixed price, number of bids received, required skills, employer rating and spending history, project deadline, and current status.

## What it extracts

- **Project Title** — Name of the freelance project.
- **Description** — Full project requirements and scope.
- **Budget** — Price range or fixed budget amount.
- **Bid Count** — Number of freelancer bids received.
- **Required Skills** — Listed technical skills needed.
- **Employer Rating** — Client's marketplace rating score.
- **Employer Spend** — Total client spending on the platform.
- **Deadline** — Project timeline or due date.

## What you can do with it

- Market rate analysis: Extract budgets across project categories to understand what clients actually pay. Compare rates for web development, mobile apps, graphic design, and other skills.
- Bid competitiveness: Bid counts reveal how competitive each project type is. High bid counts mean more freelancer competition for those project categories.
- Skill demand mapping: Extract required skills across projects to identify which technical skills have the most demand. Spot emerging skill requirements early.
- Employer quality signals: Employer ratings and total spending indicate client quality. Data helps freelancers prioritize clients who pay well and rate fairly.

## How it works

Enter the requested input when prompted. The flow opens the target page, extracts the fields listed above, and saves them as a CSV in your home folder.

## Frequently asked questions

**Can I track bidding activity over time?**

Schedule repeated extractions of the same project page to see how bid counts grow. This reveals how quickly competition increases after a project is posted.

**Does it extract individual bid amounts?**

The robot extracts publicly visible data from the project page. Individual freelancer bid amounts may or may not be publicly displayed depending on project settings.

**Can I filter projects by skill?**

Search Freelancer.com for specific skills before extracting. The robot captures whatever project page you provide.

**Does it work for contest-style projects?**

The robot extracts data from standard project pages. Contest pages have different structures and may produce different field sets.
