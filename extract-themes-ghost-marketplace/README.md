# Ghost Theme Marketplace

Ghost is a modern, open-source publishing platform used by creators, publications, and businesses who prioritize clean writing experiences and fast, SEO-friendly websites. Ghost's official theme marketplace curates professionally designed themes - both free and premium - each listing showing the theme name, developer, price, theme type, preview image, and a direct link to the theme page. Unlike WordPress's massive theme directory with tens of thousands of options, Ghost's marketplace is deliberately curated, making each listing more significant in terms of market presence.

## What it extracts

- **Position** — Ranking order of the theme in marketplace results.
- **Name** — Theme product title.
- **Brand** — Creator or studio name.
- **Price** — Free or dollar amount.
- **Type** — Theme category (blog, magazine, newsletter, etc.).
- **Image URL** — URL to the theme preview or thumbnail image.
- **Link** — URL to the theme listing or live demo page.
