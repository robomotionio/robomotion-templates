# DeBank Wallet Portfolio

DeBank is the most comprehensive DeFi portfolio tracker available, aggregating data across hundreds of protocols and dozens of blockchains into unified wallet profiles. Each DeBank profile displays a wallet's total value, token-by-token holdings across chains, liquidity pool stakes, yield farming allocations, and historical transaction data.

## What it extracts

- **Name** — Wallet identifier or label.
- **Address** — The blockchain address being tracked.
- **Total Asset** — Aggregate USD value across all chains.
- **Total Asset Percentage** — Portfolio allocation percentage by asset.
- **TVF** — Total value of funds under management.
- **Followers** — Number of followers tracking this wallet.
- **Following** — Number of wallets this address is following.
- **Earnings** — Total earnings or yield generated.
- **Hi Offer Price** — Highest offer price for assets.
- **Position** — Ranking or position in portfolio.
- **Asset Name** — Individual token or asset name.
- **Asset Value** — USD value of individual asset holdings.
- **Asset Logo** — Visual identifier for the asset.
- **Blockchain Name** — Name of the blockchain network.
- **Blockchain Asset Value** — Holdings value per blockchain.
- **Blockchain Logo** — Visual identifier for the blockchain.

## What you can do with it

- Multi-chain portfolio visibility: DeBank aggregates holdings across Ethereum, BNB Chain, Polygon, Arbitrum, Optimism, and dozens more chains. One extraction captures the complete cross-chain picture.
- Protocol position tracking: See exactly where a wallet has capital deployed - which lending pools, DEX liquidity positions, and yield farms hold their assets.
- Token holding breakdowns: Every token in a wallet with current balance and USD value. Identify concentrated positions and diversification patterns.
- Whale wallet monitoring: Track known whale addresses over time. Schedule extractions to see how large holders shift between protocols and tokens.
