# n8n Integrations

N8n is the open-source workflow automation platform that has rapidly gained traction as a self-hostable alternative to Zapier and Make. Its power comes from its node library - hundreds of pre-built connections to SaaS tools, databases, APIs, and services. For automation engineers, IT teams, and platform evaluators, the node directory is the single most important factor in deciding whether N8n can handle your automation requirements.

## What it extracts

- **Position** — The sequential position number of the integration in the N8n directory listing.
- **App Name** — The integration or app name as listed in N8n's directory.
- **App Link** — The direct URL link to the integration's page within N8n's node directory.
- **App Image** — The logo or thumbnail image representing the integration or app.

## What you can do with it

- N8n's full node catalog as structured data - every integration listed with its name, category, and capabilities for evaluation and planning.
- Platform comparison: compare N8n's node count and category coverage with Zapier, Make, and Workato to decide which automation platform fits your needs.
- Workflow feasibility assessment: before building a complex automation in N8n, verify that nodes exist for every tool in the workflow chain.
- Gap analysis: identify tool categories where N8n has few or no nodes. Plan whether custom nodes or API modules can fill those gaps.
