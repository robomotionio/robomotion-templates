# Ghost Integrations

Ghost has grown from a simple blogging platform into a full publishing and membership ecosystem - and its integration directory reflects this evolution. The directory lists tools for email marketing, analytics, e-commerce, payment processing, automation, and more. For teams evaluating Ghost as their CMS, understanding its integration ecosystem is critical.

## What it extracts

- **Position** — The rank or position of the integration in Ghost's directory listing.
- **Integration Name** — The name of the connected tool or service.
- **Integration Link** — The URL link to the integration's page or documentation.
