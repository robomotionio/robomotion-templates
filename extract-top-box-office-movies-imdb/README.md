# IMDb Box Office

IMDb's box office page is one of the most-referenced sources for tracking theatrical movie performance in the United States. Updated weekly, it displays the top-grossing films of the current weekend along with their weekend gross revenue, total cumulative gross, the number of weeks each film has been in theaters, IMDb user ratings and vote counts, movie descriptions, poster images, and week-over-week ranking changes. This data directly reflects consumer moviegoing behavior and studio marketing effectiveness.

## What it extracts

- **Position** — Current weekend ranking.
- **Movie Title** — Film name.
- **Weekend Gross** — Current weekend revenue.
- **Total Gross** — Cumulative US theatrical revenue.
- **Weeks Released** — How long the film has been in theaters.
- **IMDb Rating** — User rating score on IMDb.
- **Rating Votes** — Number of user votes for the rating.
- **Poster Image** — Movie poster thumbnail.
- **Movie Description** — Plot synopsis and film details.
- **Movie Link** — Direct link to the IMDb movie page.

## What you can do with it

- Weekend performance tracking: Extract the latest weekend gross for every ranked film. See exactly which movies are winning the box office each weekend.
- Cumulative revenue monitoring: Total gross figures show a film's complete theatrical run performance. Extract these to compare lifetime earnings across films.
- Release longevity analysis: Weeks-in-release data reveals which films have staying power. Extract this to identify movies with sustained audience interest versus front-loaded openings.
- Studio market share: Distributor data shows which studios dominate the current box office. Extract to analyze studio performance and market share trends.
