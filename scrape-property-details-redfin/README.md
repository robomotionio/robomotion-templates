# Redfin Property Details

When you find a property on Redfin that warrants serious consideration, the individual listing page reveals far more than the search results view. Each [Redfin](https://www.redfin.com/) property detail page contains the full street address, current list price, bedroom and bathroom counts, interior square footage, year built, a narrative property description written by the listing agent, home facts with key property characteristics, the Redfin estimate showing the platform's valuation, price per square foot calculation, agent commission details, school district information, walk and transit scores, estimated mortgage payments, and comparable property references. This depth of information is exactly what buyers, agents, and investors need before making an offer or investment decision.

## What it extracts

- **Address** — 123 Oak Street, San Francisco, CA 94102
- **Price** — $1,250,000
- **Beds / Baths** — 3 bed / 2 bath
- **Area-SqFt** — 1,850 sq ft
- **Redfin Estimate** — $1,265,000

## What you can do with it

- Comprehensive single-property due diligence: Extract every data point from a Redfin listing in seconds. Get address, price, specifications, tax history, and property description structured into one record without manual copying.
- Investment underwriting data: Tax history reveals assessed values and annual tax burdens over time - key inputs for investment return modeling. Extract to populate your underwriting spreadsheets without transcription errors.
- Comparative market analysis input: Appraisers and agents pull detailed specs from comparable properties. Extract property details from multiple listing pages to build accurate comparable sets efficiently.
- Price history and negotiation intelligence: Redfin shows prior list prices, price reductions, and sale history. Extract this timeline to understand a seller's negotiating position and market pricing trajectory.

## How it works

Enter the requested input when prompted. The flow opens the target page, extracts the fields listed above, and saves them as a CSV in your home folder.

## Frequently asked questions

**How is this different from the Redfin search results scraper?**

The Redfin search results scraper (page 218) extracts summary-level data from multiple listings on a search results page. This robot goes deep on a single property - extracting the complete detail page including full description, home facts, Redfin estimate, and all property specifications.

**Does this extract the full price history?**

Yes. Redfin property pages include price history showing prior list prices, price reductions, and past sale prices. The robot extracts the price history data visible on the page.

**Can I extract data for off-market or sold properties?**

Redfin maintains pages for recently sold properties. Navigate to a sold property's detail page on Redfin and paste that URL - the robot extracts whatever data the page displays.
